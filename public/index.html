Lab 0 Shenanigans

Welcome to the lab 0 Shenanigans!
There are three sections of Shenanigans
See below the URL paths and what each does!

  • /math

    URL bar Calculator

    Enter in the format: /math/number1/operation/number2

    operations include:

    • plus

    • minus

    • times

    • divideby

    • tothepowerof

  • /greet

    Enter in the format: /greet?name=kaylee&dob=2002

    This will return the name entered and how old they are!
